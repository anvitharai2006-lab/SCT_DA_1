Superstore Sales Dashboard

An interactive Excel dashboard built using Microsoft Excel to analyze sales performance, profit trends, and category-wise insights.

Features :

- Pivot Tables
- KPI Cards
- Interactive Slicers
- Conditional Formatting
- Sales Trend Analysis
- Category-wise Profit Analysis
- Data Visualization using Charts

Tools Used :

- Microsoft Excel

Project Overview :

This dashboard was created using the Superstore Sales dataset to visualize and analyze sales data effectively. The dashboard provides insights into sales trends, profit distribution, and order performance through interactive filters and visual analytics.

Key Insights :

- Sales trends over time
- Profit comparison across categories
- Region-wise performance analysis
- Interactive filtering using slicers
